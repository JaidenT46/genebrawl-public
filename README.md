## Usage

**Install required modules**:  
```npm install```   

**Build the script**:  
```npm run build_dev``` - a DEBUG version  
```npm run build``` - a PRODUCTION (FREE) version  
```npm run build_vip``` - a PRODUCTION (VIP) version  

**Or build & run it**:  
```npm run gadget``` - a DEBUG version, Gadget-mode  
```npm run gadget_prod``` - a PRODUCTION (FREE) version, Gadget-mode.  
```npm run gadget_vip``` - a PRODUCTION (VIP) version, Gadget-mode.    
```npm run frida_mac_dev``` - a DEBUG version on Mac, the game needs to be installed via PlayCover and its bundle name must be `gene.brawl.dev`*  
```npm run ios_laser``` - a PRODUCTION (FREE) version on iDevice, the game must have `com.supercell.laser`* bundle name.  
```npm run ios_laser_dev``` - a DEBUG version on iDevice, the game must have `com.supercell.laser`* bundle name.

\* You change it in the package.json file.
